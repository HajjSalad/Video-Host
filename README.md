### Video Host Repository

This repository hosts a JSON file containing a list of publicly accessible video URLs.  
It is designed to be used by Android apps or other clients that want to fetch and play videos dynamically.

## JSON Structure

The `videos.json` file has the following format:

```json
[
  {
    "title": "title 1",
    "url": "url 1"
  },
  {
    "title": "title 2",
    "url": "url 2"
  }
]
